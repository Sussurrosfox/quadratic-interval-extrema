# 二次函数闭区间最值定理

## a>0

对于二次函数 f(x) = ax² + bx + c（a > 0）在闭区间 [m,n] 上的最值问题，其完全由不变量 P = f'(m)f'(n) 和 S = f'(m) + f'(n) 的符号组决定：

**最大值**

最大值的位置完全且唯一地取决于端点导数之和 S 的符号：

* 若 S > 0，则最大值为 f(n)
* 若 S < 0，则最大值为 f(m)
* 若 S = 0，则最大值为 f(m) = f(n)

**最小值**

最小值的位置取决于端点导数之积 P 的符号：

* 情形 A：内部极小（P < 0）
导数在端点异号，区间内必有唯一驻点 x₀ 满足 f'(x₀) = 0。
* 最小值为 f(x₀)


* 情形 B：端点极小（P ≥ 0）
导数在端点同号（或有零点），函数在区间上单调。此时由 S 决定方向（注：因 f'(m) < f'(n)，此情形下 S ≠ 0）：
* 若 S > 0，则最小值为 f(m)
* 若 S < 0，则最小值为 f(n)

## a<0

对于二次函数 f(x) = ax² + bx + c（a < 0）在闭区间 [m,n] 上的最值问题，其完全由不变量 P = f'(m)f'(n) 和 S = f'(m) + f'(n) 的符号组决定。

**最小值**

最小值的位置完全且唯一地取决于 S 的符号：

- 若 S > 0，则最小值为 f(m)
- 若 S < 0，则最小值为 f(n)
- 若 S = 0，则最小值为 f(m) = f(n)

**最大值**

最大值的位置取决于 P 的符号：

- 情形 A：内部极大（P < 0）  
  导数在端点异号，区间内必有唯一驻点 x₀ 满足 f'(x₀) = 0。  
  最大值为 f(x₀)。

- 情形 B：端点极大（P ≥ 0）  
  导数在端点同号（或有零点），函数在区间上单调。此时由 S 决定方向（注：因 a<0 时 f'(m) > f'(n)，此情形下 S ≠ 0）：  
  - 若 S > 0，则最大值为 f(n)  
  - 若 S < 0，则最大值为 f(m)
