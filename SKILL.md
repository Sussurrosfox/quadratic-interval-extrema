# 二次函数闭区间最值

对二次函数 f(x)=ax²+bx+c（a≠0）在闭区间 [m,n] 上的最值问题，引入端点导数不变量：

- P = f'(m)·f'(n)
- S = f'(m)+f'(n)

其中 f'(x)=2ax+b。

## 使用流程

1. 计算端点导数 f'(m)=2am+b 与 f'(n)=2an+b。
2. 计算 P = f'(m)f'(n) 与 S = f'(m)+f'(n)。
3. 根据 a>0 或 a<0 选择对应分支。
4. 依据 P、S 的符号组直接判定最值位置，并计算对应函数值。

## 判定规则

### 当 a>0

- **最大值**：由 S 的符号唯一决定。
  - S>0：最大值为 f(n)
  - S<0：最大值为 f(m)
  - S=0：最大值为 f(m)=f(n)
- **最小值**：由 P 的符号决定。
  - P<0：内部极小，最小值为 f(x₀)，其中 f'(x₀)=0
  - P≥0：端点极小
    - S>0：最小值为 f(m)
    - S<0：最小值为 f(n)

### 当 a<0

- **最小值**：由 S 的符号唯一决定。
  - S>0：最小值为 f(m)
  - S<0：最小值为 f(n)
  - S=0：最小值为 f(m)=f(n)
- **最大值**：由 P 的符号决定。
  - P<0：内部极大，最大值为 f(x₀)，其中 f'(x₀)=0
  - P≥0：端点极大
    - S>0：最大值为 f(n)
    - S<0：最大值为 f(m)
